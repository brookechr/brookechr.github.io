Braintrust Slack Bot

It is a Slack bot that allows users of the Work on Climate Slack workspace of around 1,500 people to search terms and have their synonyms come up in the results, rather than just exact string matching. We are also working on implementing a chat bot that proactively messages users the contact information of people with similar backgrounds. I've been contributing to this project since December 2020.

I am working on the project voluntarily with a group of industry developers and an industry product manager. As the only student in the group, it has been a valuable learning experience to work with such experienced and helpful people.

We are using Python, Airtable, Google App Engine, and GitHub for this project. I set up the AirTable for the definitions and synonyms, which are crowd-sourced. A lot of the other work that I've done has been in the form of pair programming over video calls.
